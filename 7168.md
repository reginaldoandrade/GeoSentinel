# 🔎 Criminal Record Search – How to Use

Follow these steps to use the **Crime Record Search** feature in Geo Sentinel.

## Steps

1. Go to the **top-right corner** of the interface.
2. You will see the **Search** option.
3. **Hover over the search icon**.
4. A small menu will appear with options.
5. Enable the **first checkbox**.
6. Once enabled, the **Crime Record Search popup** will open.
7. You can now **search for crime records** using the available fields.
